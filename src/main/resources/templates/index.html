Currency converter project

This is test currency converter project where you can take data from EU bank and convert money from different currencies.

Used technologies:

    Java 11
    Maven
    Spring boot 2.3.2
    PostgresSQL 12 (Docker)
    HTML/CSS/Thymeleaf

About

The application was written in java spring boot. Used Database was PostgresSQL 12 which was running on Docker. The main Dependencies used spring-web, thymeleaf, JPA and e.t.c
